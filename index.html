Let's play Rock, Paper, Scissors!

Reglas del juego

Jugaras contra una IA.

Puedes elegir entre piedra, papel o tijera.

El primero que gane 3 rondas gana.

Aqui estan las normas para ganar una ronda:

  • Piedra gana a tijeras
  • Tijeras gana a papel
  • Papel gana a piedra

Si el jugador y la IA sacan lo mismo (ej. ambos sacan papel), entonces nadie se lleva el punto.

Puntuación de jugador: 0 Puntuación de IA: 0
